### 이클립스 단축키 모음
- Java 개발하면서 자주 사용했던 이클립스 단축키를 정리합니다.

#### 주석
- 한줄 주석 : ctrl + /
- 블록 주석 : 영역 선택 후 ctrl + shift + /

#### 소스
- 한줄 복사 : ctrl + alt + ↓
- 한줄 이동 : alt + ↓ ( ↑ )

- Open Type : ctrl + shift + T
- Open Resource : ctrl + shift + R
- 세로 편집 모드 적용/해제 : alt + shift + A

- Format : ctrl + shift + F
- Organize Imports : ctrl + shift + O

- Rename : alt + shift + R
