jQuery 를 이용하여 XML의 하위노드에 데이터를 추가하기 위한 방법이다.

{% highlight javascript %}
var xmlData = $.parseXML( "<root></root>" );
var _xml = $.parseXML( "<test></test>" );
$(xmlData).children(0).append($(_xml).children(0));
{% endhighlight %}

xmlData, _xml 이 xml object 인 경우 생략해도 된다.
